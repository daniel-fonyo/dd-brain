The Tools

Two agentic skills, built on Claude Code

/review-pr

Parallel multi-dimensional code review

  • 10 dimensions reviewed simultaneously
  • Searches Confluence for applicable rules
  • Self-verifies: reads source code, writes failing tests
  • Posts as PENDING review. Human submits.

/sandbox-test

Autonomous E2E sandbox testing

  • Engineer describes what to test, agent handles how
  • Drives real browser, captures 5 signal sources
  • 3 independent loads, cross-load comparison
  • Runs in background. Engineer is free.

The Meta-Point

Skills = Encoded Standards

  • Tribal knowledge becomes executable. 15+ Confluence docs nobody reads mid-review. Now they're checklists that run automatically.
  • Consistency without fatigue. Friday at 5pm gets the same quality as Monday at 10am.
  • Compound learning. Each correction persists. Each Glean discovery is saved. The skill never makes the same mistake twice.
  • Fine-grained control. Each dimension is a markdown file. Add a check = edit one file. No deployment.
  • Human judgment where it matters. Agent does the exhaustive scan. You make the call.
